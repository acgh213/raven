# ✧ raven — goals ✧

## v0.1 — a reader that works

- [ ] import an OPML and add/remove a feed URL
- [ ] poll RSS/Atom safely and politely
- [ ] retain source HTML and extract readable content
- [ ] expose a cursor-paginated river and article API
- [ ] run durable jobs through restart/retry
- [ ] sync idempotent reading events
- [ ] ship an Android Compose reader backed by Room
- [ ] read cached articles completely offline
- [ ] restore a backup into a working service

## v0.2 — useful organization

- [ ] provider registry and health-aware background fallback
- [ ] summaries, controlled tags, entities, versioned embeddings
- [ ] conservative clusters with visible confidence and sibling headlines
- [ ] hand-authored interest profile and explainable relevance sort

## v0.3 — contextual conversation

- [ ] article/cluster/source/passage anchored chat sheet
- [ ] server-owned context assembly and streaming protocol
- [ ] live model availability and explicit per-conversation selection
- [ ] reviewable memory candidates

## v0.4 — accountable agency

- [ ] proposed profile changes with provenance, approve/reject/revert
- [ ] standing watches with explainable matches
- [ ] timezone-aware editions with thin-window suppression
- [ ] in-app edition history and deep links

## later

- desktop or web client against the same API
- export sinks (email, Nextcloud, static HTML)
- source folders and richer feed organization
- highlights, annotations, share-to-subscribe flow
- local notifications for watch hits
- opt-in persona-bearing Hermes entries

## non-goals

- a synthesized pane that replaces the feed
- opaque model filtering or deletion
- a mandatory cloud account
- a public news service
- multi-user permissions in v0.1
- making model use a prerequisite for basic reading
- building four editions full of filler because a clock happened to ring
